Behavior notes

  • Always one empty row at the bottom — typing in it creates the next empty row automatically. No "+ Add line" button needed.

  • Type toggle color switches the total bar color (red for expense, green for income) and the submit button color.
  • Payment method applies to the whole document (not per line) — one payment method per entry.
  • No note field — description on each line replaces the old single note field.
